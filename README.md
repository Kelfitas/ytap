# YTAP - YouTube Audio Player

# Requirements
- mpv

# Install
1. Clone: `git clone https://github.com/Kelfitas/ytap.git`
1. Run `sudo make`

# Hotkeys
- ctrl+k: show debug
- ctrl+m: show menu
- ctrl+F4: prev video
- ctrl+F5: play/pause video
- ctrl+F6: next video
